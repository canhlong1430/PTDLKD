Thực hiện tiền xử lý dữ liệu: làm sạch, chuẩn hóa và biến đổi dữ liệu.

Phân tích dữ liệu mô tả và khám phá (EDA).

Ứng dụng các phương pháp phân tích/thống kê để đánh giá xu hướng, hiệu quả kinh doanh.

Trực quan hóa dữ liệu bằng biểu đồ nhằm hỗ trợ trình bày kết quả phân tích.

Tổng hợp kết quả và đề xuất nhận định phục vụ ra quyết định.
